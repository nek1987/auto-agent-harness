# Nano Banana Pro Style Keywords Reference

Vocabulary for crafting effective prompts organized by category.

---

## Multi-Image Reference Keywords

### Image Notation
- "Image 1", "Image 2", "Figure 1", "Figure 2"
- "The uploaded image", "The reference image"
- "First image", "Second image"

### Element Selection
- "from Image 1", "from Image 2"
- "the person in Image 1"
- "the outfit from Image 2"
- "the pose in Figure 2"
- "the lighting from Image 2"
- "the color palette from Figure 2"

### Combination Instructions
- "Combine", "Merge", "Blend"
- "Apply [X] from Image 1 to Image 2"
- "Transfer", "Copy", "Use"
- "Keep [element] from [source]"
